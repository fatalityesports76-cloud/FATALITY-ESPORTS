Privacidade

Última atualização: 10/02/2026.

1. Dados Coletados

Durante o recrutamento e acesso interno, podemos coletar dados de identificação, contato e informações competitivas necessárias para avaliação e operação da organização.

2. Uso dos Dados

Os dados são utilizados para triagem de candidatos, comunicação oficial, autenticação de membros e segurança do ambiente interno.

3. Base de Segurança

Aplicamos controles técnicos, incluindo validações de acesso, trilhas de auditoria e mecanismos de proteção para reduzir risco de uso indevido.

4. Compartilhamento

A Fatality não comercializa dados pessoais. Compartilhamentos ocorrem apenas quando estritamente necessários para operação técnica do site ou cumprimento legal.

5. Retenção

Dados são mantidos pelo período necessário para recrutamento, gestão de conta e segurança operacional, com descarte quando não houver mais finalidade válida.

6. Direitos do Titular

O usuário pode solicitar atualização ou revisão de dados de conta pelos canais oficiais, respeitando limites de segurança e regras internas da organização.
